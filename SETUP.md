# Инструкция по установке и запуску

## Быстрый старт

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите dev-сервер:**
   ```bash
   npm run dev
   ```

3. **Откройте браузер:**
   ```
   http://localhost:3000
   ```

## Структура проекта

- `/app` - страницы Next.js (App Router)
- `/components` - React компоненты
- `/lib` - утилиты, типы и мок-данные
- `/public` - статические файлы

## Основные страницы

- `/` - Главная страница с KPI и картой
- `/impact` - Интерактивная карта мероприятий и метрики
- `/judge-pack` - Генератор Judge Pack (PDF)
- `/outreach` - Мероприятия и аутрич
- `/resources` - Ресурсы и материалы
- `/mentor` - Mentor Hub
- `/media` - Медиа галерея
- `/evidence` - Evidence Vault
- `/news` - Новости/блог
- `/about` - О команде

## Особенности

### Judge Pack Generator
- Автоматическая генерация PDF резюме
- Выбор ключевых мероприятий
- Экспорт для судей

### Impact Map
- Интерактивная карта на Leaflet
- Фильтры по типу и году
- Экспорт в CSV

### Impact Dashboard
- Графики и метрики
- Визуализация данных

### Система новостей
- Добавление новостей
- Теги и категории
- Динамические страницы

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **React Leaflet** - карты
- **Recharts** - графики
- **jsPDF** - генерация PDF

## Следующие шаги

1. Подключите базу данных (Supabase, PostgreSQL, MongoDB)
2. Добавьте аутентификацию (NextAuth.js)
3. Настройте загрузку файлов (Cloudinary, AWS S3)
4. Добавьте email-уведомления
5. Настройте деплой (Vercel, Netlify)

## Примечания

- Мок-данные находятся в `/lib/data.ts`
- Типы определены в `/lib/types.ts`
- Для работы карты нужен клиентский рендеринг (используется dynamic import)
